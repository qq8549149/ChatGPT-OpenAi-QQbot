设置温度+小数

说明：例如设置温度0.8 代表随机性高

设置字数+整数
说明：例如设置字数1000 代表机器人最多回复500个汉字

切换问答模式
说明：问答模式不记忆上下文 可用于写代码

切换对话模式
说明：对话模式记忆上下文 可用于调教

设置人设+文本
说明：修改默认的人设

重置
说明：清空聊天内容

回滚
说明：删除，字数超过时，回滚的的对话次数

打开ai或者关闭ai

说明：不需要艾特就可以和机器人对话
